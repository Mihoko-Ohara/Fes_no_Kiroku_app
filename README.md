# 概要
「フェスのきろく」はフェスやライブに行った思い出や情報を投稿できる共有ブログです。自分が行ったフェスやライブについて記録し、みんなで共有することで今まで知らなかったバンドやフェスに出会うことが出来ます。

# 機能一覧

- ブログ投稿機能
- キーワード検索機能
- ブログの削除機能
- ブログの詳細ページ

#　技術一覧

- Vue.js
- Laravel
- PostgreSQL
- heroku
